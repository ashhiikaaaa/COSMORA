🌌 Cosmora — Begin Beyond Earth

Cosmora is a futuristic space travel web interface that simulates booking interplanetary missions. It features an immersive UI with animated visuals, interactive navigation, and a multi-page experience including authentication, dashboard, missions, and booking flow.

🚀 Features

* 🌠 Stunning space-themed UI with animations (aurora, meteors, orbit effects)
* 🧭 Multi-page navigation (Hero, Dashboard, Destinations, Missions, Booking, etc.)
* 🔐 Login & Signup interface with modern glassmorphism design
* 📊 Dashboard with mission stats and quick actions
* 🪐 Interactive destination and mission browsing
* 🎟️ Seat selection and booking system UI
* 💳 Payment simulation and confirmation flow
* 🎯 Fully responsive design for different screen sizes


🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Styling:** Custom CSS (Glassmorphism, Animations, Gradients)
* **Fonts:** Google Fonts (Outfit, JetBrains Mono)
* **JavaScript:** (for navigation & interactivity if added separately)
